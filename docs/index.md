# **Optimización Numérica y Algoritmos Bioinspirados**

## **Descripción General**
Este proyecto constituye un estudio integral sobre la aplicación de algoritmos bioinspirados y técnicas de optimización numérica para la resolución de problemas complejos. El trabajo se divide en tres ejes fundamentales:

1.  **Optimización de Funciones Matemáticas:** Evaluación del desempeño de algoritmos estocásticos y deterministas frente a paisajes de error desafiantes (Rosenbrock y Rastrigin).
2.  **Problemas de Logística Real:** Aplicación de metaheurísticas para la optimización de rutas y costos de transporte en un escenario geográfico real (Ruta de las 32 capitales de México).
3.  **Inteligencia Artificial:** Implementación y análisis de modelos avanzados para la toma de decisiones basada en datos.

El objetivo principal es contrastar la eficiencia computacional (número de evaluaciones) frente a la precisión global en la búsqueda de soluciones óptimas.
